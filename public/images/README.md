# Images Directory

Place your image files in this directory. The following images are referenced in the components:

## Required Images:

- **kiosk-main.png** - Main kiosk image for Hero section (fallback: ./kiosk.png)
- **kiosk-1.png** through **kiosk-8.png** - Gallery images for ImageGallery component

## Image Specifications:

- Format: PNG, JPG, or WebP
- Recommended size: 1200x800px or larger for best quality
- The gallery component will automatically handle responsive sizing

## Notes:

- Images in the `public` folder are served from the root path
- Use `./images/filename.png` in components to reference images here
- Broken images will be hidden automatically by the gallery component
